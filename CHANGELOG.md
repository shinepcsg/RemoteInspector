# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-04-06

### Added
- Scene hierarchy browser with real-time search
- GameObject and component inspector
- Field editing for common serializable types (Vector3, Color, Rect, Bounds, etc.)
- Nested field and collection element editing
- Real-time log streaming with stack trace
- Remote command console with extensible custom commands
- Embedded browser UI served by the runtime
- Runtime status overlay and control panel
- Optional HTTPS/WSS transport with PFX or generated self-signed certificate
- Runtime status toggle UI (F2 key)

### Changed
- Initial public release
